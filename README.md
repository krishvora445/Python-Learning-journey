# Python Learning Journey

This repository contains my Python practice files, notes, and mini-projects while I learn Python.

## About this repository

The goal of this repo is to keep all my Python learning materials in one place, organized by topic and chapter. It includes:

- basic Python syntax
- variables and data types
- type casting and user input
- comments, printing, and escape sequences
- operators and conditionals
- loops and strings
- functions and modules
- lists, tuples, sets, and dictionaries
- object-oriented programming
- advanced Python concepts
- file handling and external modules
- AI-related practice code
- small Python projects

## How to use

1. Open any folder that matches the topic you want to practice.
2. Run the `.py` file inside it.
3. Modify the examples and experiment with your own code.
4. Keep adding new notes, exercises, and projects as you progress.
